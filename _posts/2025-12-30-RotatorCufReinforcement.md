主要记录一下自己左肩在引体的时候弹响问题解决的思考

### 问题描述

引体在最低端完全下放之后再启动，明显感觉左肩会有卡顿感，右肩没有问题

### 可能原因

总的原因大概就三类，一个个筛查就完了

引体向上底端（肩完全伸展）时出现声音，常见可能性包括：

1. 肩胛骨下沉或上旋不够
   + 底端时肩胛骨应该“上旋 + 上提 + 外展”，如果你在底端时松掉肩胛或让肩被拉得过度向上，会让肩袖被牵拉。
2. 肩袖（尤其是冈下肌/小圆肌）在极端位置稳定不足
   + 底端位置需要保持肱骨头居中，弱的话会出现轻微位移感、摩擦感。 
3. 前锯肌或下斜方肌弱，无法维持肩胛节律
  + 这会让你底端肩胛“塌”下去，导致上臂骨和肩胛不协调。
4. 关节结构原因：肩峰形态、轻微撞击等
   + 只要不痛、力量正常，一般通过训练能改善。

如果不下放到底就没声音，这说明你的症状在“最大牵拉 + 肩胛失控”这个组合出现，典型是肩胛稳定不足 + 肩袖控制不够强

### 特异性分析

1. 侧卧外旋，分析那边偏弱->冈下肌或小圆肌偏弱
2. 靠墙滑动，分析耸肩跟卡顿->前锯肌或肩胛控制偏弱
3. YTW观察，分析那边控制弱->肩胛稳定性弱

### 训练

以上每一个我都有问题...

因此我的训练目标分三部分了

1. 提升肩袖稳定
2. 让肩胛在“引体向上底端”能正确上旋、上提、外展
3. 让引体动作更顺畅、避免关节摩擦

所以我的强化就针对以下几个部分，这种小肌群不用大重量训练，主要是高频次、控制好动作质量

1. 肩袖强化（冈下肌、小圆肌）
   1. 侧卧哑铃外旋（最有效的冈下肌练法）
   2. Prone W（俯卧 W）
2. 前锯肌
   1. 靠墙滑动
   2. 熊爬
3. 肩胛下沉与下斜方肌训练(引体最低端的控制)
   1. Scapular pull-up（肩胛引体），不要弯手，只做肩胛“上提-下沉”
   2. Y Raise（俯卧 Y）
   3. Farmer carry with shoulder depression，农夫行走

### 训练计划

我现在的安排是

| 星期 | 内容                     |
| ---- | ------------------------ |
| 周一 | Sqaut day，主深蹲 + 腹部 |
| 周二 | 推日，主要是卧推类       |
| 周三 | 拉日，主要是硬拉         |
| 周四 | 休息                     |
| 周五 | 小蹲日                   |
| 周六 | 小推日 主实力推          |
| 周日 | 休息                     |

我在小推日就主要安排肩袖训练，同时在推日着重热一下前锯肌，拉日安排肩胛下沉跟斜方肌的训练

因此整合的计划就是

| 星期     | 主练科目             | **训前激活 (3-5 min)**                        | **训练中/后辅助 (5 min)**                              | 针对性目的                                                      |
| :------- | :------------------- | :-------------------------------------------- | :----------------------------------------------------- | :-------------------------------------------------------------- |
| **周一** | **主深蹲 + 腹部**    | **Y-Raise (空手)**<br>2组 x 15次              | **侧卧哑铃外旋**<br>2组 x 12次                         | 深蹲需要上背部极度张力，侧卧外旋预防杠铃压迫引起的肩部僵硬。    |
| **周二** | **推日 (卧推类)**    | **弹力带外旋**<br>2组 x 15次                  | **墙面滑动 (Wall Slide)**<br>3组 x 10次 (插在卧推组间) | 卧推易导致圆肩和撞击，利用墙面滑动抵消胸肌收缩带来的肩胛前倾。  |
| **周三** | **拉日 (死鱼/引体)** | **肩胛引体 (Scapular Pull-up)**<br>3组 x 10次 | **弹力带面拉 (Face Pull)**<br>3组 x 15次               | **重点：** 解决你左肩引体弹响。先学会用肩胛启动，再用背部发力。 |
| **周四** | **休息日**           | *全身拉伸*                                    | -                                                      | 修复筋膜，缓解前三天的累积压力。                                |
| **周五** | **小蹲日**           | **T-Raise (俯身T字)**<br>2组 x 15次           | **直臂下压 (轻重量聚焦肩胛)**<br>2组 x 15次            | 强化后侧链与肩胛下沉的能力。                                    |
| **周六** | **小推日 (实力推)**  | **弹力带 W 字伸展**<br>2组 x 15次             | **哑铃 L-Fly (L字外旋)**<br>2组 x 12次                 | 实力推（OHP）极度依赖肩袖稳定性，L-Fly 能模拟高位的稳定性。     |
| **周日** | **休息日**           | *执行晚间松解计划 + 胸大肌拉伸*               | -                                                      | 为下周一的大强度深蹲储备肩部活动度。                            |


另外每晚可以做10分钟的肩部训练

| 时间         | 环节           | 动作内容                           | 组数/时长  | 重点提示                                                 |
| :----------- | :------------- | :--------------------------------- | :--------- | :------------------------------------------------------- |
| **0-2 min**  | **快速松解**   | **筋膜球/网球松解胸小肌 & 冈下肌** | 每侧 1min  | 重点按压锁骨下方和肩胛骨后方，有痛点就停 10 秒。         |
| **2-4 min**  | **前锯肌激活** | **四足位推撑 (Bear Reach)**        | 15次 x 2组 | 双手撑地，像猫一样把上背部顶到最高，感觉腋下发力。       |
| **4-7 min**  | **肩袖力量**   | **侧卧哑铃(或矿泉水)外旋**         | 15次 x 3组 | 左侧先做。大臂夹紧身体，像转轴一样旋转，不要用躯干借力。 |
| **7-9 min**  | **肩胛功能**   | **墙壁滑动 (Wall Slide)**          | 12次 x 2组 | 后脑勺、后背、手肘、手背全程贴墙，上下滑动。             |
| **9-10 min** | **神经重置**   | **死虫式 (Dead Bug)**              | 10次       | 核心收紧，配合深呼吸，进一步稳定躯干与肩带的联系。       | **** |
