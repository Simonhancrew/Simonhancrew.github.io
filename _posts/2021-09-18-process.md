第二题开始用的组合数，其实没必要

```
LL C[N][N];
void init(){
    for(int i = 0;i < N;i++){
        for(int j = 0;j <= i;j++){
            if(!j) C[i][j] = 1;
            else C[i][j] = (C[i - 1][j] + C[i - 1][j - 1]);
        }
    }
}
```

第三题是一个二分的问题，考虑数据范围之后比较自然的就能想到

```
while(l < r){
    LL mid = l + r >> 1;
    if(check(mid)) r = mid;
    else l = mid + 1;
}
```
