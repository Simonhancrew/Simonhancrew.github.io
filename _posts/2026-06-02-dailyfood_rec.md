简单的记录一下最近的饮食，我早饭基本是不吃的，大概10点会吃第一顿练前的，中午13点吃第二顿，晚上19点吃第三顿，中间可能有夜宵

### 正常饮食模式大致描述

之前我的饮食模式是

| 时间段 | 饮食内容 |
| --- | --- |
| 10:00 | 鸡蛋（1个） + 坚果 +辅酶 Q10 100mg + 复合B(类似OTC，早晨吃着玩) + 鱼油（训练日 1g / 休息日 0.5g）|
| 13:00 | 鸡肉 + 菠菜 + 黑豆 + 糙米饭 + 苹果碎 + 海苔碎 + 一点辣椒酱（量产小袋） + 玉米 |
| 19:00 | 鸡肉（周二周四换成牛肉） + 西兰花 + 黑豆 + 番茄碎 + 奇亚籽 + 一点辣椒酱（量产小袋） + 豆腐 |

其中，周二 / 四 / 六早餐多加：D3 2500IU+K2-MK7 50μg（同鱼油脂溶共服）

下午3点我会吃一碗酸奶 + 新鲜葡萄，晚上8点左右我会喝250ml的全脂奶

晚上睡前我会吃200mg的甘氨酸镁 + 10mg的锌。

### 问题

我是正常人饭量，整个看下来，其实我的胆碱跟维生素C是摄入不够的，别的维生素看下来是够的，微量元素矿物质看起来ok。

其次看来，蛋白质在早饭是有点不够的，练前也是没有碳水补充的，练后的碳水我估计是不够的，因为糙米饭的碳水密度不高，而且我吃的属实也不是很多
