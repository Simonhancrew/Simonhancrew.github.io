蛋白质是由氨基酸组成的生物大分子，是构成身体组织、酶、激素等的重要物质基础。它们在细胞结构、代谢调节、免疫功能等方面发挥着关键作用。

人体成分中，水占比约60%，蛋白质占16-20%，脂肪占10-25%，矿物质占4-5%。蛋白质是身体中仅次于水的第二大成分，具有重要的生理功能。

### 1 蛋白质元素

主要元素：

+ 碳（C）：构成蛋白质的骨架和氨基酸的侧链。
+ 氢（H）：存在于氨基酸的氨基（-NH2）和羧基（-COOH）中。
+ 氮（N）：氨基酸的氨基部分含有氮，是蛋白质的特征元素。
+ 氧（O）：存在于氨基酸的羧基中，参与蛋白质的结构和功能。
+ 硫（S）：存在于某些氨基酸（如半胱氨酸和蛋氨酸）中，参与蛋白质的三维结构稳定。

六大营养素包含：碳水化合物(CHO)、脂肪(CHO)、蛋白质(CHONS)、维生素、矿物质和水。蛋白质是三大宏量营养素之一，提供4千卡/克的能量，是身体组织修复和生长的关键营养素。且蛋白质是唯一含氮的宏量营养素（除了某些矿物质和维生素含极少氮），这使得它在身体代谢中具有独特的作用，如氮平衡、肌肉合成和免疫功能等。

### 2 如何判断食物中是否含有蛋白质

判断食物中是否含有蛋白质可以通过以下几种方法：

#### 1. **查看营养标签**
食品包装上的营养成分表会明确标注蛋白质含量（以克为单位），通常以"蛋白质"或"Protein"字样显示。

#### 2. **了解食物种类**
- **动物来源**：肉类、鱼类、蛋类、奶制品（牛奶、酸奶、奶酪）
- **植物来源**：豆类（黄豆、黑豆、红豆）、坚果、种子、全麦面包、燕麦

#### 3. **化学检测法**

##### 凯氏定氮法（Kjeldahl Method）
凯氏定氮法是一种经典的蛋白质含量测定方法，由丹麦化学家约翰·凯耶达尔（Johan Kjeldahl）于1883年发明。该方法基于以下原理：

**原理**：将样品中的蛋白质与浓硫酸和催化剂一起加热分解，使蛋白质中的氮转化为硫酸铵，然后通过碱化蒸馏将铵态氮转化为氨气，用硼酸溶液吸收后，再用标准酸溶液滴定，从而计算出样品中的含氮量，再乘以蛋白质换算系数（通常为6.25）得到蛋白质含量。

**优点**：
- 测定结果准确可靠
- 适用于各种蛋白质含量的样品
- 是国际标准方法（ISO 5983）

**局限性**：
- 只能测定总氮含量，不能区分蛋白质氮和非蛋白质氮
- 检测过程较复杂，需要专业设备
- 分析时间较长

**相关事件**：
2008年中国发生的**三聚氰胺奶粉事件**就是利用了凯氏定氮法的局限性。不法分子在奶粉中添加三聚氰胺（含氮量高达66%），通过提高总氮含量来虚假提高蛋白质检测值，严重危害了婴幼儿健康。

##### 其他检测方法
- **双缩脲试剂法**：蛋白质与双缩脲试剂反应产生紫色络合物，用于定性检测
- **酚试剂法**：与蛋白质中的酪氨酸和色氨酸反应，产生蓝色物质
- **紫外分光光度法**：利用蛋白质在280nm波长处的吸收特性进行测定

#### 4. **口感和质地**
富含蛋白质的食物通常有独特的口感，如肉类的嚼劲、豆腐的嫩滑、鸡蛋的凝固特性。

### 3 氨基酸结构

氨基酸是蛋白质的基本构成单位，其基本结构通式如下：

```
      H
      |
  H2N-C-COOH
      |
      R
```

- **氨基（-NH₂）**：碱性基团，含有氮元素，是蛋白质的特征性基团
- **羧基（-COOH）**：酸性基团，可电离出氢离子
- **α-碳（C）**：连接氨基和羧基的碳原子
- **R基团**：侧链基团，不同氨基酸的R基团不同，决定了氨基酸的性质

### 4 氨基酸，肽，蛋白质的关系

氨基酸通过**肽键**连接形成肽和蛋白质：

1. **氨基酸**：构成蛋白质的最小单位（单体）
2. **肽键**：一个氨基酸的羧基与另一个氨基酸的氨基脱水缩合形成的化学键（-CO-NH-）
3. **二肽**：2个氨基酸通过1个肽键连接而成
4. **寡肽**：3-10个氨基酸组成的肽链
5. **多肽**：10个以上氨基酸组成的长链
6. **蛋白质**：由一条或多条多肽链折叠、盘曲形成的具有特定空间结构和功能的生物大分子

**简单关系图**：氨基酸 → 肽键连接 → 肽 → 空间折叠 → 蛋白质

### 5 氨基酸的分类

氨基酸是构成蛋白质的基本单位，人体内有20种标准氨基酸。根据人体是否能自身合成以及合成的能力，氨基酸可以分为以下三类：

#### 1. 必需氨基酸（Essential Amino Acids, EAA）

人体**无法自身合成**，必须通过食物摄入的氨基酸，共9种（成人8种，儿童9种）：

| **氨基酸** | **功能与特点** | **食物来源** |
| :--------: | :------------: | :----------: |
| 赖氨酸 | 促进生长发育，增强免疫力，改善记忆力 | 肉类、鱼类、蛋类、豆类、坚果 |
| 色氨酸 | 合成血清素和褪黑素，调节睡眠和情绪 | 牛奶、鸡蛋、肉类、鱼类、香蕉 |
| 苯丙氨酸 | 合成多巴胺和肾上腺素，改善情绪 | 肉类、鱼类、蛋类、豆类 |
| 甲硫氨酸 | 参与解毒作用，保护肝脏 | 肉类、鱼类、蛋类、坚果 |
| 苏氨酸 | 参与胶原蛋白合成，促进伤口愈合 | 肉类、鱼类、蛋类、奶制品 |
| 亮氨酸 | 促进肌肉蛋白质合成，抗疲劳 | 肉类、鱼类、蛋类、豆类 |
| 异亮氨酸 | 参与能量代谢，维持血糖平衡 | 肉类、鱼类、蛋类、坚果 |
| 缬氨酸 | 促进伤口愈合，调节血糖 | 肉类、鱼类、蛋类、豆类 |
| 组氨酸 | 儿童生长发育必需，调节免疫系统 | 肉类、鱼类、蛋类、豆类 |

#### 2. 非必需氨基酸（Non-essential Amino Acids）

人体**可以自身合成**，不需要完全依赖食物摄入。共有11种：

| **氨基酸** | **功能与特点** | **食物来源** |
| :--------: | :------------: | :----------: |
| 甘氨酸 | 参与胶原蛋白合成，促进伤口愈合 | 肉类、鱼类、蛋类、豆类 |
| 丙氨酸 | 参与葡萄糖代谢，提供能量 | 肉类、鱼类、蛋类、豆类 |
| 丝氨酸 | 参与磷脂合成，保护细胞膜 | 肉类、鱼类、蛋类、坚果 |
| 半胱氨酸 | 抗氧化作用，保护细胞 | 肉类、鱼类、蛋类、豆类 |
| 酪氨酸 | 合成甲状腺激素和黑色素 | 肉类、鱼类、蛋类、奶制品 |
| 天冬氨酸 | 参与尿素合成，解毒作用 | 肉类、鱼类、豆类、蔬菜 |
| 天冬酰胺 | 参与蛋白质合成，调节代谢 | 肉类、鱼类、豆类、坚果 |
| 谷氨酸 | 合成谷胱甘肽，抗氧化 | 肉类、鱼类、豆类、谷类 |
| 谷氨酰胺 | 参与免疫调节，修复肠道 | 肉类、鱼类、豆类、奶制品 |
| 脯氨酸 | 参与胶原蛋白合成，维持结构 | 肉类、鱼类、蛋类、豆类 |
| 精氨酸 | 合成一氧化氮，扩张血管 | 肉类、鱼类、蛋类、坚果 |

#### 3. 条件必需氨基酸（Conditionally Essential Amino Acids）

在正常情况下人体可以合成，但在**特定生理状态**（如疾病、创伤、生长发育期）下，合成能力不足，需要额外补充。主要包括：

| **氨基酸** | **功能与特点** | **食物来源** |
| :--------: | :------------: | :----------: |
| 精氨酸 | 在创伤、手术或严重感染时需要额外补充 | 肉类、鱼类、蛋类、坚果 |
| 半胱氨酸 | 在早产儿或某些疾病状态下需要补充 | 肉类、鱼类、蛋类、豆类 |
| 酪氨酸 | 在苯丙酮尿症患者中需要补充 | 肉类、鱼类、蛋类、奶制品 |
| 谷氨酰胺 | 在高强度运动或疾病恢复阶段需要补充 | 肉类、鱼类、豆类、奶制品 |

### 6 蛋白质的功能

蛋白质在人体中发挥着多种重要功能：

#### 酶

蛋白质作为酶（生物催化剂），参与体内几乎所有的化学反应：
- 催化效率高：比无机催化剂效率高10^7-10^13倍
- 专一性强：一种酶通常只催化一种或一类反应
- 调节代谢：如淀粉酶催化淀粉水解，蛋白酶催化蛋白质分解

#### 结构

蛋白质是构成身体结构的重要成分：
- 肌肉组织：肌动蛋白和肌球蛋白组成肌肉纤维
- 骨骼和牙齿：胶原蛋白和骨钙蛋白参与骨骼形成
- 皮肤和结缔组织：胶原蛋白和弹性蛋白维持皮肤弹性

#### 运输功能

蛋白质参与物质的运输和传递：
- 血红蛋白：在红细胞中运输氧气和二氧化碳
- 脂蛋白：运输脂肪和胆固醇
- 转铁蛋白：运输铁元素

#### 免疫防御

蛋白质是免疫系统的重要组成部分：
- 抗体（免疫球蛋白）：识别和中和病原体（细菌、病毒）
- 补体系统：增强免疫细胞的吞噬作用
- 白细胞介素：调节免疫细胞的活性

#### 信号传递

蛋白质作为信号分子参与细胞间通信：
- 激素：如胰岛素（调节血糖）、生长激素（促进生长发育）
- 受体：细胞膜上的蛋白质受体接受信号分子
- 转录因子：调控基因表达

#### 水盐平衡

蛋白质参与维持体内的水盐平衡，蛋白能锁住管内液的水，血管内的蛋白流失，会导致水分流向间质导致水肿：
- 血浆蛋白（如白蛋白）维持渗透压
- 离子通道蛋白控制离子进出细胞
- 水通道蛋白调节水分运输

#### 运动功能

蛋白质参与肌肉收缩和运动：
- 肌动蛋白和肌球蛋白：构成肌肉纤维，参与收缩
- 肌钙蛋白和原肌球蛋白：调节肌肉收缩过程
- 运动相关酶：提供肌肉收缩所需能量

#### 能量功能

蛋白质可以作为能量来源，包括糖异生 + 参加三羧酸循环：
- 每克蛋白质提供4千卡能量（与碳水化合物相同）
- 在长期饥饿或低碳水化合物饮食时，蛋白质会分解提供能量
- 蛋白质的能量功能通常是次要的，主要功能是结构和代谢调节

### 7 蛋白质的分类

根据蛋白质的营养价值和氨基酸组成，蛋白质可以分为以下几类：

#### 完全蛋白

**定义**：含有人体所需的所有必需氨基酸，且比例适当，能满足人体需要

**特点**：
- 氨基酸模式与人体蛋白质模式接近
- 生物价高，消化吸收率好
- 能完全满足人体生长发育和维持健康的需要

**食物来源**：
- 动物来源：鸡蛋、牛奶、肉类、鱼类、酸奶
- 植物来源：大豆蛋白（豆腐、豆浆、毛豆）、藜麦、螺旋藻

#### 半完全蛋白

**定义**：含有所有必需氨基酸，但某些氨基酸比例不适当，不能完全满足人体需要

**特点**：
- 氨基酸组成不平衡，通常缺乏1-2种必需氨基酸
- 生物价较低，消化吸收率较差
- 单独食用可能无法满足生长发育需求

**食物来源**：
- 植物来源：小麦、大麦、燕麦、玉米、大米、花生

#### 不完全蛋白

**定义**：缺乏一种或多种必需氨基酸，不能单独满足人体蛋白质需求

**特点**：
- 氨基酸组成不完整，缺乏关键必需氨基酸
- 生物价低，单独食用无法维持氮平衡
- 需要与其他蛋白质互补食用

**食物来源**：
- 植物来源：明胶（缺乏色氨酸）、玉米胶蛋白（缺乏赖氨酸）
